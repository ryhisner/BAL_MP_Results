These files contain the results of the BAL mutation pattern (MP) analysis, which describes the mutation pattern characteristic of bronchoalveolar lavage-sample sequences from chronic infections.

The 'Positions' files contain the results from considering only the positions at which amino acid mutations take place.

The 'Substitutions' files contain the results when analyzing specific amino acid substitutions.

The Clades, Pangos, and Megaclade files list the number of sequences with the BAL MP that belong to a given Pango lineage, Nextstrain Clade, or Megaclade.

Megaclades combine related clades. The Megaclades: pre-VOC, Alpha, Beta, Gamma, Delta, BA.1, BA.2, BA.4/5, XBB, BA.2.86, and recombinant. Details for the precise clades included in each Megaclade are listed in a table included in the spreadsheet.
